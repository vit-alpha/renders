vit — the cluster Δ 1RM cell (output construction)

Graph 7 of the series. Traces one number — the base Δ cell in the vit review overview table — from raw sets to the printed value.

Reading the graph. One number, left to right: raw sets on the cluster's base_exercise are filtered by observed_points(), curve() picks a recency-decayed 1RM winner per window, the winner's own laterality factor scales it to bilateral-equivalent kg, and Δ = newer − older lands in the base Δ cell of the vit review overview table. The shaded band marks the stages that run twice — once per comparison window.

Side branches (dashed): when a window has no direct observations on the base, related-exercise observations are projected through the relationships graph and the median projected 1RM fills the window (_window_rm1 in queries/cluster_progress.py). The flags box is a separate curve_diff() call harvested only for shape_mismatch — the Δ printed here comes from the two per-window curve() results, not from curve_diff.delta_rm1, so curve_diff's min-observations gate does not apply to this cell.
